# Branderly Creative Publisher

Website landing page resmi untuk **Branderly Creative Publisher**, penerbit e-book akademik yang berfokus pada dosen dan civitas akademika. Kami membantu menerbitkan buku ajar, modul, monograf, dan buku referensi di berbagai bidang ilmu seperti kesehatan, teknik, pertanian, ekonomi, pendidikan, dan sosial humaniora.

## Fitur Utama
- **Landing Page Responsif**: Dibangun dengan HTML dan Tailwind CSS untuk tampilan optimal di desktop dan mobile.
- **Section Lengkap**: Hero, Tentang, Bidang Ilmu, Keunggulan, Alur Penerbitan, Kontak, dan Footer.
- **Fokus Akademik**: Dirancang khusus untuk mendukung peningkatan nilai KUM dosen melalui publikasi berkualitas.
- **Mudah Dishare**: Hosting gratis via GitHub Pages.
